## Build Locally

0. Install [Node.js](https://nodejs.org) and [Yarn](https://yarnpkg.com).

1. Clone:

    ```bash
    git clone https://github.com/gear-tech/wiki.git
    cd wiki
    ```

2. Install dependencies:

    ```bash
    yarn install
    ```

3. Serve and open in the browser (http://localhost:3000/):

    ```bash
    yarn run start
    ```
